# 📰 AI-агент: Telegram-канал новин України

Агент автоматично веде Telegram-канал: читає стрічку [Укрнету](https://www.ukr.net/news/main.html),
відбирає найважливіші новини, пише текст поста через AI (Google Gemini, безкоштовно),
додає фото з першоджерела або згенеровану обкладинку і публікує в канал.

## Як це працює

1. **GitHub Actions** запускає скрипт кожні 30 хвилин (безкоштовно, комп'ютер не потрібен).
2. Скрипт читає стрічку Укрнету і бачить, скільки видань написали про кожну подію —
   це сигнал важливості (наприклад, «394 публікації» = головна подія дня).
3. **Розклад** (київський час):
   - 🌞 06:00–01:00 — пост кожні 30 хв (гарячі новини) або 60 хв (звичайні)
   - 🌙 01:00–06:00 — раз на 2 год (гарячі) або 3 год (звичайні)
4. Для обраної новини агент знаходить статтю-першоджерело, бере з неї опис і фото,
   а Gemini пише унікальний текст поста. Якщо фото немає — генерується обкладинка з заголовком.
5. Пост публікується з посиланням на першоджерело. Стан (що вже опубліковано)
   зберігається в `state.json`, дублі відсіюються навіть якщо новина оновилась.

## Налаштування (один раз, ~15 хвилин)

### Крок 1. Створіть канал і бота

1. У Telegram створіть канал (Новий канал → назва, опис, аватарка).
2. Напишіть боту [@BotFather](https://t.me/BotFather): `/newbot` → назва → username бота.
   Скопіюйте **токен** (виглядає як `123456:ABC-DEF...`).
3. Додайте бота **адміністратором** каналу з правом «Публікація повідомлень»
   (Канал → Адміністратори → Додати адміністратора → знайдіть бота за username).

### Крок 2. Отримайте безкоштовний ключ Gemini

1. Відкрийте [aistudio.google.com/apikey](https://aistudio.google.com/apikey) (потрібен Google-акаунт).
2. Натисніть **Create API key** і скопіюйте ключ (`AIza...`).

Безкоштовного тарифу вистачає з великим запасом (агенту треба ~30–50 запитів на день).
Без ключа агент теж працює — просто пости будуть без AI-тексту (заголовок + опис + посилання).

### Крок 3. Розгорніть на GitHub

1. Створіть **приватний** репозиторій на [github.com](https://github.com/new).
2. Завантажте туди цей проєкт:
   ```bash
   git init
   git add .
   git commit -m "Новинний агент"
   git branch -M main
   git remote add origin https://github.com/ВАШ_ЛОГІН/ВАШ_РЕПОЗИТОРІЙ.git
   git push -u origin main
   ```
3. У репозиторії: **Settings → Secrets and variables → Actions → New repository secret**.
   Додайте три секрети:
   | Назва | Значення |
   |---|---|
   | `TELEGRAM_BOT_TOKEN` | токен від BotFather |
   | `TELEGRAM_CHANNEL` | `@username_каналу` |
   | `GEMINI_API_KEY` | ключ з AI Studio |
4. Вкладка **Actions** → увімкніть workflows, якщо GitHub спитає.
5. Перший пост: **Actions → «Пост новин у Telegram» → Run workflow** (можна поставити
   галочку «force», щоб запостити одразу). Далі все працює саме.

> ⚠️ GitHub автоматично вимикає розклад, якщо в репозиторії 60 днів немає комітів.
> Наш workflow сам комітить `state.json` після кожного поста, тому це не проблема.

## Локальний запуск (тест на своєму ПК)

```powershell
pip install -r requirements.txt
copy .env.example .env       # і впишіть свої ключі в .env
python -m newsbot.main --dry-run --force   # показати пост, нічого не публікуючи
python -m newsbot.main --force             # опублікувати прямо зараз
```

## Налаштування під себе

Усе в [newsbot/config.py](newsbot/config.py): інтервали дня/ночі, поріг «гарячої» новини
(`HOT_THRESHOLD`), максимальний вік новини, кількість постів за запуск.
Стиль текстів — промпт у [newsbot/llm.py](newsbot/llm.py).
Дизайн обкладинки — [newsbot/cover.py](newsbot/cover.py).

## Важливо про авторські права

Укрнет — агрегатор: він збирає заголовки інших ЗМІ. Агент чинить так само коректно:
пише **власний** короткий переказ (через AI) і завжди ставить посилання на першоджерело.
Не вимикайте посилання на джерело — це і етично, і захищає канал від скарг.
